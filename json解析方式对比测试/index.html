json解析方式对比测试

2024年3月7日

解析方法代码

编写Benchmark，启动测试

测试结果

结论：

  1. 标准库json解析最慢，运行时间6159ns，内存占用最多1312B
  2. 采用对象池，内存占用下降明显30%，但是执行时间下降7%
  3. 采用Jsoniter，内存占用下降明显30%，执行时间下降70%。
  4. 采用对象池+Jsoniter，内存占用下降60%，执行时间下降80%
  5. json解析频繁情况下，可以采用对象池+Jsoniter
